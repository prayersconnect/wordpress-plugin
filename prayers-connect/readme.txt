== Description ==

Prayers Connect WordPress plugin provides miscellaneous functionalities from https://prayersconnect.com. The plugin currently includes a shortcode to easily link a mosque to Prayers Connect.

== Installation ==

1. Upload the `prayers-connect` folder to the `/wp-content/plugins/` directory.
2. Activate the "Prayers Connect" plugin through the 'Plugins' menu in WordPress.
3. You can now use the `[prayers_connect_link]` shortcode in your WordPress posts or pages.

== Shortcode Usage ==

The `[pc_mosque_link]` shortcode allows you to link a mosque to Prayers Connect. It accepts the following attributes:

- `mosqueId`: The ID of the mosque.
- `title`: The title of the link.

Example usage: `[pc_mosque_link id="123" title="Mosque Link"]`

== Frequently Asked Questions ==

= How can I link a mosque to Prayers Connect? =

To link a mosque to Prayers Connect, use the `[pc_mosque_link]` shortcode in your WordPress posts or pages. Provide the mosque ID and the desired title as attributes in the shortcode.

= Can I customize the appearance of the mosque link? =

Yes, you can use custom CSS to style the link according to your needs.
